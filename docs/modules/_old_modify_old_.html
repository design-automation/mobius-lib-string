substring

  • substring(str: string, start: number, end: number): string
    • Defined in old/modify-old.ts:68

    Extracts characters in a string between two specified indices and returns it as a new string. Start index is inclusive and end index is exclusive.

    example

    string = "Orange" substring = str.Modify.substring(string,1,4)

    Expected value of substring is "ran".

    Parameters

    • str: string

      String.

    • start: number

      Zero-based index to start extracting characters (inclusive).

    • end: number

      Zero-based index to stop extracting characters (exclusive).

    Returns string

    New string with extracted characters.
